# Become a Validator

Any Rubix node is eligible to become a validator (or "miner"). To qualify as a validator in the open Rubix network, a node must hold enough pledgeable RBT tokens to pledge against the tokens in a transaction it validates. A minimum of one validator is required to validate a transaction, and the quorum size can be increased.
